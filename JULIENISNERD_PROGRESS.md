# JULIENISNERD Portfolio - Project Progress & Status

## 1. 프로젝트 개요 (Overview)
- **프로젝트명**: JULIENISNERD Portfolio
- **목적**: 브루탈리즘/미니멀리즘 감성의 인더스트리얼 디자인 포트폴리오 웹사이트 구축
- **기술 스택**: HTML5, CSS3 (Vanilla), JavaScript (ES6+), GSAP (ScrollTrigger)
- **현재 테마**: Light Theme (화이트 배경, 블랙 텍스트 통일 적용)
- **사용 폰트**: Inter (우아하고 얇은 네오 그로테스크 스타일)

## 2. 파일 및 구조 현황 (File Structure)
- `index.html`: 메인 갤러리 페이지 (가로 스크롤 레이아웃)
- `project.html`: 개별 프로젝트 상세 페이지 (세로 스크롤 및 패럴랙스 효과)
- `contact.html`: 연락처 정보 페이지
- `admin.html`: 프로젝트 업로드용 숨겨진 관리자 페이지 (비밀번호: 0000)
- `style.css`: 전체 웹사이트 공통 스타일 (Light Theme 적용 완료, 텍스트 컬러 #000 통일)
- `project.css`: 상세 페이지 전용 스타일
- `main.js`: 메인 페이지 GSAP 애니메이션 및 동적 렌더링 로직
- `project.js`: 상세 페이지 URL 파라미터 기반 데이터 렌더링 및 애니메이션
- `admin.js`: 관리자 페이지 기능 (로컬 스토리지 연동, 프로젝트 추가/수정/삭제 CRUD 완비)
- `data.js`: 전역 프로젝트 데이터 (기본 제공 프로젝트 3개 및 로컬 스토리지 연동 함수)

## 3. 핵심 기능 구현 상태 (Key Features)

### ✅ 완료된 작업 (Completed)
1. **Light Theme & Black Text 통일**
   - 기존 어두운 배경을 제거하고, 전체 `#FAFAFA` ~ `#FFFFFF` 배경에 텍스트 색상을 완벽한 검은색(`#000`)으로 통일함.
2. **언어 영문화 적용**
   - 시연용 프로젝트(Speaker, Camera, Chair)의 소개글과 에러 메시지를 모두 영어로 번역 및 적용함.
3. **가로/세로 복합 스크롤 시스템**
   - 메인 페이지는 마우스 휠을 통해 가로로 부드럽게 스크롤되며, 상세 페이지는 세로형으로 이미지 패럴랙스(시차) 애니메이션 적용.
4. **로컬 스토리지(Local Storage) 기반 어드민 시스템**
   - 백엔드 서버 없이 브라우저 저장소를 활용하여 누구나 웹페이지 내에서 즉각적으로 프로젝트를 임시 등록하고 확인할 수 있는 구조 확립.
5. **어드민 CRUD 완성 (Edit / Delete 기능 추가)**
   - 기존 프로젝트 목록을 우측 패널에 렌더링 (썸네일 + 메타데이터 포함).
   - Edit 버튼: 클릭 시 해당 프로젝트 데이터가 좌측 폼에 자동 채워지고 "UPDATE PROJECT" 모드로 전환.
   - Delete 버튼: 커스텀 모달 확인 후 삭제, 목록 즉시 반영.
   - 2-컬럼 레이아웃 (좌: 폼, 우: 목록), 이미지 URL 실시간 미리보기 기능 포함.
   - data.js에 updateProject(), deleteProject() 함수 추가로 CRUD 완비.
6. **프로젝트 이미지 다중 슬라이드 캐러셀**
   - 프로젝트 상세 페이지에서 좌/우 버튼, 도트 인디케이터, 드래그/터치/휠/키보드 화살표로 여러 이미지 탐색 가능.
   - 데이터 구조: `images[]` 배열 사용 (없으면 기존 `img` 단일 이미지로 폴백).
   - 어드민에서 이미지 URL 다중 입력 지원 (+ Add Image URL 버튼, 미니 미리보기, 삭제 버튼).
   - 첫 번째 이미지가 갤러리 썸네일(`img`)로 자동 설정.

### ⏳ 향후 계획 (Upcoming & Backlog)

   - 필요 시 Firebase/Supabase 기반의 데이터베이스 연동으로 완벽한 CMS(웹 관리 시스템) 구축.
2. **모바일 반응형(Responsive) 디테일 개선**
   - 가로 스크롤 레이아웃에 대한 모바일/태블릿 UI 최적화 보강.
